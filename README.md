About This Repo

This repository tracks my preparation for the Capital One Senior Data Analyst CodeSignal assessment. Problems are modelled after real Capital One assessment question types: CSV-based data analysis and SQL queries set in financial services business scenarios.

The CodeSignal DAA consists of:



14 quiz questions — CSV data analysis using a tool of your choice (Excel, Google Sheets, Python/pandas)

1 SQL coding question — written directly in the CodeSignal IDE

70 minutes total
